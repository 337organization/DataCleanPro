#### 2.1.2 文件导入面板

实现了文件选择、导入、预览功能：

```java
public class ImportPanel extends JPanel {
    private JButton selectFileButton;
    private JButton importButton;
    private JTable dataTable;
    private DefaultTableModel tableModel;
    private JLabel fileInfoLabel;
    private JFileChooser fileChooser;
    
    private void initComponents() {
        selectFileButton = new JButton("选择文件");
        importButton = new JButton("导入数据");
        importButton.setEnabled(false);
        
        tableModel = new DefaultTableModel(columns, 0);
        dataTable = new JTable(tableModel);
        
        fileChooser = new JFileChooser();
        fileChooser.setFileFilter(new FileNameExtensionFilter(
            "支持的文件 (*.xlsx, *.xls, *.csv)", "xlsx", "xls", "csv"));
    }
    
    private void setupEventHandlers() {
        selectFileButton.addActionListener(e -> selectFile());
        importButton.addActionListener(e -> importFile());
    }
    
    private void selectFile() {
        int result = fileChooser.showOpenDialog(this);
        if (result == JFileChooser.APPROVE_OPTION) {
            selectedFile = fileChooser.getSelectedFile();
            fileInfoLabel.setText("已选择: " + selectedFile.getName());
            importButton.setEnabled(true);
        }
    }
}
```

### 2.2 JTable数据表格

实现了动态列和数据展示：

```java
private void updateTable(List<DataRow> rows) {
    // 清空表格
    tableModel.setRowCount(0);
    
    if (rows.isEmpty()) {
        return;
    }
    
    // 动态更新列名
    DataRow firstRow = rows.get(0);
    if (firstRow.getFields() != null) {
        String[] columns = new String[firstRow.getFields().size()];
        for (int i = 0; i < firstRow.getFields().size(); i++) {
            columns[i] = "列 " + (i + 1);
        }
        tableModel.setColumnIdentifiers(columns);
    }
    
    // 添加数据行
    for (DataRow row : rows) {
        if (row.getFields() != null) {
            tableModel.addRow(row.getFields().toArray());
        }
    }
}
```

### 2.3 异步任务处理

使用SwingWorker实现异步操作，避免GUI冻结：

```java
private void importFile() {
    if (selectedFile == null) {
        JOptionPane.showMessageDialog(this, "请先选择文件", 
            "提示", JOptionPane.WARNING_MESSAGE);
        return;
    }
    
    // 使用SwingWorker在后台执行导入
    SwingWorker<DataFile, Void> worker = new SwingWorker<>() {
        @Override
        protected DataFile doInBackground() throws Exception {
            return importService.importFile(selectedFile);
        }
        
        @Override
        protected void done() {
            try {
                DataFile dataFile = get();
                if (dataFile != null) {
                    fileInfoLabel.setText("导入成功: " + 
                        dataFile.getFileName() + 
                        " (" + dataFile.getRowCount() + " 行)");
                    loadDataPreview(dataFile.getId());
                    JOptionPane.showMessageDialog(ImportPanel.this, 
                        "文件导入成功！", "成功", 
                        JOptionPane.INFORMATION_MESSAGE);
                }
            } catch (Exception e) {
                JOptionPane.showMessageDialog(ImportPanel.this, 
                    "文件导入失败: " + e.getMessage(), 
                    "错误", JOptionPane.ERROR_MESSAGE);
            }
        }
    };
    
    worker.execute();
}
```

### 2.4 状态栏实现

实现了显示连接状态和进度的状态栏：

```java
public class StatusBar extends JPanel {
    private JLabel messageLabel;
    private JProgressBar progressBar;
    private JLabel connectionLabel;
    
    public StatusBar() {
        setLayout(new BorderLayout());
        setBorder(BorderFactory.createEtchedBorder());
        
        messageLabel = new JLabel("就绪");
        progressBar = new JProgressBar(0, 100);
        progressBar.setStringPainted(true);
        connectionLabel = new JLabel("数据库: 未连接");
        
        JPanel leftPanel = new JPanel(new FlowLayout(FlowLayout.LEFT));
        leftPanel.add(messageLabel);
        
        JPanel centerPanel = new JPanel(new FlowLayout(FlowLayout.CENTER));
        centerPanel.add(progressBar);
        
        JPanel rightPanel = new JPanel(new FlowLayout(FlowLayout.RIGHT));
        rightPanel.add(connectionLabel);
        
        add(leftPanel, BorderLayout.WEST);
        add(centerPanel, BorderLayout.CENTER);
        add(rightPanel, BorderLayout.EAST);
    }
    
    public void setConnectionStatus(boolean connected) {
        if (connected) {
            connectionLabel.setText("数据库: 已连接");
            connectionLabel.setForeground(Color.GREEN);
        } else {
            connectionLabel.setText("数据库: 未连接");
            connectionLabel.setForeground(Color.RED);
        }
    }
}
```

## 三、遇到的问题与解决方案

### 3.1 GUI冻结问题

**问题描述：** 文件导入时GUI界面会冻结，无法响应用户操作。

**解决方案：** 使用SwingWorker在后台线程执行耗时操作：
```java
SwingWorker<DataFile, Void> worker = new SwingWorker<>() {
    @Override
    protected DataFile doInBackground() {
        // 在后台线程执行
        return importService.importFile(file);
    }
    
    @Override
    protected void done() {
        // 在EDT线程更新GUI
        updateUI(get());
    }
};
worker.execute();
```
