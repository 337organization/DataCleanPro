# DataCleanPro 演示说明

## 演示环境准备

### 1. 数据库准备
1. 启动 MySQL 服务
2. 执行 `src/main/resources/db/schema.sql` 创建数据库和表
3. 修改 `application.properties` 中的数据库配置

### 2. 编译项目
```bash
mvn clean compile
```

### 3. 运行项目
```bash
mvn exec:java -Dexec.mainClass="com.datacleanpro.App"
```

## 演示要点

### 1. OOP 特性展示
- **继承**：FileParser 抽象类 → ExcelParser、CsvParser
- **接口**：DataCleaner 接口 → DuplicateCleaner、EmptyValueCleaner、FormatCleaner
- **多态**：ParserFactory 根据文件类型自动选择解析器

### 2. 多线程展示
- 文件导入时使用 SwingWorker 异步执行
- 进度条显示任务进度
- GUI 不会冻结

### 3. 网络编程展示
- Socket 服务器监听端口
- 客户端发送请求
- JSON 格式数据交换

### 4. 数据库操作展示
- 连接池管理
- 批量数据插入
- 事务处理

### 5. 文件处理展示
- Excel 文件解析（Apache POI）
- CSV 文件解析（OpenCSV）
- 文件格式自动识别

## 常见问题解答

### Q: 数据库连接失败怎么办？
A: 检查 MySQL 服务是否启动，验证 `application.properties` 中的数据库配置是否正确。

### Q: 文件导入失败怎么办？
A: 检查文件格式是否支持（.xlsx, .xls, .csv），确认文件编码是否为 UTF-8。

### Q: 导出报表失败怎么办？
A: 检查 `storage/reports/` 目录是否存在且有写入权限。

### Q: 如何查看详细日志？
A: 查看 `storage/logs/` 目录下的日志文件。
